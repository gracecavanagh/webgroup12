ArcGIS Instant Apps

ArcGIS Instant app is a web based application which provides number of options to a user to configure the app and to show the Mapping data in 2D as well as 3D. It can also have specific templates for viewing scene, comparing content, exploring gallery of contents etc.

Advantages Disadvantages
Highly customizable Can clutter a small screen if there are too many pop-up windows
Separate view for mobile devices available to configure Not configured for landscape view
Compatible with 3D Scenes Data needs to be ArcGIS Online compatible
Displays location-based data and interactive maps on the same screen Reliant on internet connection
No coding needed Large amounts of data and layers may cause the webpage to crash on a mobile device.
