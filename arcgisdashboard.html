ArcGIS Dashboard

An ArcGIS Dashboard is a way of presenting location information and relative data on the same screen, so that everything you need to make a decision or monitor conditions are right in front of you at once. Optionally, you can use data from Esri Living Atlas like the NOAA (National Oceanic and Atmospheric Administration) along with items from your content on ArcGIS Online.

Advantages Disadvantage
Highly customizable Can clutter a small screen if there are many of data windows
Separate view for mobile devices available to configure Not configured for landscape view
Compatible with 3D Scenes Data needs to be ArcGIS Online compatible
Displays location-based data and interactive maps on the same screen Reliant on internet connection
No coding needed The ArcGIS user-friendly interface can sometimes be limiting, but does offer opportunities to add our own code.
